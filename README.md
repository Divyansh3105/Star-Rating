# Star Rating

A clean and interactive star rating component built with HTML, CSS, and JavaScript. This project demonstrates how to create a responsive and visually appealing rating system for web applications.

## Features
- Interactive star selection
- Responsive design
- Customizable styles
- Easy integration into any project

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```
2. **Open `index.html` in your browser**

## Usage
- Integrate the `index.html`, `style.css`, and `script.js` files into your project.
- Customize the styles in `style.css` as needed.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
